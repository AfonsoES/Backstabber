## Detalhes do jogo
### Oque é o Backstabber
O Back stabber é um RPG de Ação baseado nos primos do Roberto sendo o Alberto o vilão pricipal que colocou um artifacto nas costas de todos os primos menos do Roberto para os podere controlar.
### Objetivo
O player jogo com uma personagem que chama-se Ivy, ela veio do mundo normal, mas caiu em uma dolina para este mundo.<br>
O objetivo é tirar o artifacto das costas de todos os primos do Roberto e derrutar o Alberto.
